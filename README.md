# FuelFast-project

# Overview

FuelFast is a simple, static website inspired by Uber, designed to allow users to order fuel (petrol, diesel, or hydrogen) for delivery to their location. The design is classic and orderly, featuring a clean grid layout, serif typography, and classic styling. This project is aimed solve Automotive issues

# Features

Fuel Selection: Users can choose from Petrol, Diesel, or Hydrogen.
Order Form: A form appears to input quantity and delivery location after selecting a fuel type.
Classic Design: Uses a timeless aesthetic with serif fonts and a neutral color palette.
Basic Interactivity: Form submission triggers an alert with order details (for demo purposes).

# Files

index.html: The main structure of the website, including header, fuel options, order form, and footer.
index.css: Styles the website with a clean, classic look using a grid layout and responsive design.
index.js: Adds interactivity, such as showing the order form and handling form submissions with a simple alert.

# Setup

<!-- Open Locally -->

Open index.html in a web browser (e.g., Chrome) by double-clicking it or dragging it into the browser.

<!-- Test -->

Click "Order Now" on a fuel type to see the form.
Fill out the form and submit to trigger an alert with your order details.

# Usage

Browse the fuel options in the grid.
Select a fuel type to open the order form.
Enter quantity and location, then submit to see a confirmation alert.
This is a static demo; no data is saved or sent.
